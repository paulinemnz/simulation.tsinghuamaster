# Stakeholder Portrait Images

Place the stakeholder portrait images in this directory with the following filenames:

1. **emma-thalman.png** - Emma Thalman (CFO)
2. **milo-gergiev.png** - Dr. Milo Gergiev (Head of Engineering)
3. **david-werner.png** - David Werner (Procurement Director)
4. **laura-moreau.png** - Laura Moreau (Head of HR)
5. **ig-metall.png** - IG Metall Chapter (Union Representative)

## Image Requirements:
- Format: PNG or JPG
- Size: Square images recommended (e.g., 200x200px or larger)
- Style: Professional headshot/portrait photos
- Background: Transparent or solid color background

If images are not found, the system will automatically fall back to displaying initials in circular avatars.
