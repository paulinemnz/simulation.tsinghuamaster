# Image Assets

Please place your image files here:

1. **sem-logo.png** - The Tsinghua SEM logo (purple circular emblem with star and Chinese characters)
2. **robot-image.png** - The futuristic robot image (thinking pose, white matte finish, blue glowing eyes)

## File Requirements:
- **sem-logo.png**: Should be a square image, preferably 200x200px or higher resolution
- **robot-image.png**: Should be a portrait-oriented image, preferably 600x800px or higher resolution

## Supported Formats:
- PNG (recommended for logos with transparency)
- JPG/JPEG
- SVG (if you have vector versions)

Once you place the images in this folder, the landing page will automatically use them.
